# 賞鳥探索冒險遊戲 - 專案總覽

## 🎮 專案簡介

這是一個基於神經科學研究《The Tuned Cortex》的賞鳥探索網頁遊戲。遊戲結合了認知訓練與娛樂性，讓玩家在虛擬環境中體驗賞鳥的樂趣，同時訓練多種認知功能。

### 核心特色

✨ **基於科學研究**
- 根據神經可塑性研究設計
- 訓練視覺搜索、模式識別、工作記憶等認知功能
- 呼應專家與新手的大腦差異

🐦 **真實的賞鳥體驗**
- 40+ 種台灣本地鳥類
- 5 個不同的棲息地環境
- 真實的鳥類行為模擬

🎯 **豐富的遊戲內容**
- 探索模式：自由探索各種棲息地
- 任務系統：30+ 個挑戰任務
- 成就系統：50+ 個成就徽章
- 認知訓練：5 種小遊戲

📚 **教育意義**
- 學習鳥類知識
- 了解生態保育
- 培養觀察能力

## 🎨 遊戲特色

### 1. 探索系統
- **多樣化棲息地**：都市公園、森林、濕地、山區、海岸
- **動態環境**：時間、天氣、季節變化
- **自由探索**：玩家可自由移動和觀察

### 2. 觀察機制
- **望遠鏡**：遠距離觀察鳥類
- **相機**：拍攝高品質照片
- **錄音機**：記錄鳥鳴聲音
- **筆記本**：記錄觀察心得

### 3. 識別挑戰
- **難度分級**：從簡單到困難
- **本地 vs 非本地**：呼應研究中的熟悉度概念
- **特徵記錄**：訓練觀察細節
- **即時回饋**：學習正確識別

### 4. 圖鑑系統
- **詳細資訊**：包含學名、特徵、習性
- **觀察記錄**：追蹤發現歷史
- **統計數據**：收集進度和成就
- **比較功能**：並排比較相似物種

### 5. 認知訓練
- **記憶配對**：訓練工作記憶
- **找不同**：訓練精細識別
- **快速識別**：訓練反應速度
- **聲音配對**：訓練聽覺識別
- **視覺搜索**：訓練空間注意力

## 🎓 研究背景

本遊戲的設計靈感來自以下研究：

**論文**: The Tuned Cortex: Convergent Expertise-Related Structural and Functional Remodeling across the Adult Lifespan

**作者**: Erik A. Wing, Jordan A. Chad, Geneva Mariotti, Jennifer D. Ryan and Asaf Gilboa

**主要發現**:
1. 賞鳥專家的大腦在注意力和感知相關區域密度更高
2. 專家在識別不熟悉鳥類時展現不同的神經活動模式
3. 賞鳥活動可能有助於保持老年大腦健康
4. 專業技能的培養會在大腦結構上留下印記

**遊戲應用**:
- 設計「本地」vs「非本地」物種的識別挑戰
- 實作多種認知訓練小遊戲
- 追蹤玩家的識別準確率和進步
- 提供適應性難度調整
