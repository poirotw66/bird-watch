## 🎮 當前可玩功能

### 遊戲操作
1. **移動**：使用 WASD 或方向鍵移動玩家
2. **觀察鳥類**：
   - 靠近鳥類（150 像素內）
   - 按住 F 鍵或滑鼠左鍵開始觀察
   - 觀察 5 秒以上獲得最高識別率
   - 再次按 F 鍵結束觀察
3. **拍攝照片**：
   - 靠近鳥類（200 像素內）
   - 按 C 鍵拍照
   - 距離越近分數越高
4. **查看圖鑑**：
   - 按 P 鍵開啟圖鑑
   - 上下鍵選擇鳥類
   - 查看詳細資訊和統計
   - ESC 關閉
5. **查看任務**：
   - 按 Q 鍵開啟任務列表
   - 左右鍵切換分頁（進行中/可接取/已完成）
   - 上下鍵選擇任務
   - Enter 接受任務
   - ESC 關閉
6. **查看資訊**：
   - 左上角顯示玩家資訊
   - 右上角顯示控制說明
   - 左下角顯示系統狀態
   - 右側顯示成就通知

### 遊戲機制
- **觀察品質**：根據距離和時長計算
  - 優秀：距離 < 30m，時長 ≥ 5s
  - 良好：距離 < 60m，時長 ≥ 3s
  - 差：其他情況
- **拍照評分**：0-100 分
  - 距離評分：30 分
  - 光線品質：20 分
  - 對焦品質：30 分
  - 構圖品質：20 分
- **圖鑑解鎖**：
  - 常見鳥類：觀察 1 次
  - 不常見：觀察 1 次 + 拍照 1 次 + 識別成功
  - 稀有：觀察 2 次 + 良好照片 1 張 + 識別成功
  - 傳說：觀察 3 次 + 優秀照片 2 張 + 識別成功
